# Automated API Testing Platform

A backend project for practicing automated API testing, CI pipelines, and containerization.

## Tech Stack
- Python
- FastAPI
- PyTest
- httpx
- Docker
- GitHub Actions

## Features
- REST API endpoints
- Automated API tests
- Health check endpoint


Important Commands:
venv\Scripts\activate

backend:
docker-compose up
docker-compose up --build
docker-compose down
runs on http://localhost:8000

frontend:
npm run dev
runs on http://localhost:5173
